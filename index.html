解释

执行 tomcat 的启动脚本 startup.sh 在执行过程中有如下调用过程：

  1. startup.sh 调用 catalina.sh；
  2. catalina.sh 调用 setclasspath.sh。

在调用过程中，如果未在环境变量中指明 jre 的路径（本文提到的配置 Java 方法实际上仅有 JDK，但是 java 命令确实可以执行）会首先在 startup.sh 中寻找是否有 JRE_HOME 的定义，如果没有再依次查询 catalina.sh 和 setclasspath.sh 。因此将 JRE_HOME 的定义放置再 setclasspath.sh 是终极解决方案。

HTTP协议

超文本传输协议（HTTP，HyperText Transfer Protocol）是互联网上应用最为广泛的一种网络协议。

简介

  • 超文本：表明通过 HTTP 可以传输的资源不止包括文本，这也是能在互联网上查看图片的基础；
  • 传输：所谓“上网”，不过是将本地请求上传，服务器资源下载；
  • 协议：客户端连接服务器后，如果想要获取服务器中的资源，需要遵守一定的通讯格式，HTTP 协议定义了客户端和服务器的通讯格式。
