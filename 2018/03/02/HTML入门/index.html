HTML入门

发表于 2018-03-02 | 分类于 前端基础 |

HTML是HyperText Markup Language的缩写，原义为超文本标记语言。HTML是网页的框架，可以理解为人体的脊柱，网页上的资源包括字符串、媒体文件和超链接等都是通过HTML代码指定和标记的。

特点

  1. 由标签来定义的语言，代码都是由标签组成；

  2. 不区分大小写；

  3. HTML文档主要由四个主要标记构成，这四个主要标记分别是<!doctype>、<html>、<head>、<body>。

    • <!doctype>文档类型声明，用于告知浏览器使用哪一种HTML规范；

    • <html>所有的HTML文档都是以<html>开头、</html>结束，并没有实际性的功能；

    • <head>用于给网页添加一些辅助或者属性的信息；

    • <body>内部为网页内容。

<head>内部标签

<title>

用于指定网页标题，该标签内容会在浏览器上显示

<meta>

该标签提供关于HTML文档的元数据，不会在用户浏览网页时显示，但是对于机器是可读的。通常位于<head>内<title>之前。

<meta>的主要属性有：

  • 网页编码格式charset选择网页内容的编码格式

  • name属性与对应的值（可选操作）：

    • 网页说明Description用于描述网页的主要内容和主题

    • 网页关键字Keywords用于描述网页内容的关键字

    • 编写网页用到的工具Genkerator用于说明编写网页时使用的工具

    • 作者Author用于说明网页制作者

  • http-equiv

    • 网页自动跳转Refresh，也可用于实现自动刷新

<body>

<body>的主要属性有：

  • 页边距margin

    • topmarigin
    • lefemarigin
    • rightmarigin

    • bottommarigin

      这四个属性用于设置网页的页边距

  • 文字颜色text 该属性用于设置网页中所有文字的默认颜色，这一属性可以对网页中所有未单独设置颜色的文本生效。

  • 超链接颜色link 设置网页中所有超链接的颜色（默认情况下未访问的超链接是蓝色，访问过的为暗红色），同样对未单独定义过颜色的超链接生效。

  • 网页背景色bgcolor 设置网页背景色。

  • 网页背景图background 设置网页背景图。
