JavaScript基础语法

发表于 2018-03-24 | 分类于 前端基础 |

JavaScript（此后也简称做js）身为一门编程语言，有其自己的语法规则。相比Java等其他程序设计语言，js的语法相对松散、严谨性较差，导致js代码更加灵活、编写难度降低，但是为了避免错误、提高可读性，在写js代码时应该想办法提高其严谨性。比如每一个语句后的分号，适当的换行和缩进等。

数据类型

JavaScript是一门弱类型的脚本语言，数据的类型不必首先声明，而是先将其声明为var类型的变量，然后在使用和赋值时再确定其数据类型。

基本数据类型

js中有四种基本数据类型：

  1. 数值型：int+float
  2. 字符串型：单引号或双引号括起来的一个或多个字符
  3. 布尔型
  4. 空值
  5. 未定义值：undefined

常量

同其他编程语言相同，js也含有基本数据类型对应的四种常量值。其中：

  1. 布尔型与C/C++相同，如果参与算术运算，false的值为0，true的值为1；
  2. 同样含有转义字符；
  3. 使用单引号或双引号括起来的一个或多个字符并没有本质的不同，都是字符串型的常量。

作用域

js中变量同样有相对应的作用域，分为全局变量和局部变量。定义在函数体内定义的变量是只能在函数体内存在的局部变量，其他的全部为全局变量。另外，js不支持语句块中的局部变量。

运算符

这三者与java基本相同。

比较运算符

由于js弱类型的特点，比较运算符与java略有不同并且加入了绝对等于（===）和不绝对等于两个特殊的比较运算符。

语句

流程控制语句

js同样有顺序、选择、循环三种流程。

选择

js的选择语句同样有if(判断条件){执行语句}和switch(){case 1; case 2...; case n; default}两种。其中：

  • if()语句允许在判断条件处填入非逻辑值（即写入赋值语句不会发生语法错误），导致js的if()语句的判断条件是错误的高发地带；
  • switch语句的条件可以写入任何类型的值。

循环

js支持的循环语句有do{循环体}while(循环条件)、while(循环条件){循环体}、for(语句1;循环条件;语句2){循环体}。请不要轻易在循环体中使用alert()函数弹出消息框 。

注释

注释语句的重要性相信不需要我在赘述了，js支持的注释有单行注释：//注释语句和多行注释/*注释语句*/两种。
