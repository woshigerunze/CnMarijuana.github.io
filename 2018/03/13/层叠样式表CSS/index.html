层叠样式表CSS

发表于 2018-03-13 | 分类于 前端基础

层叠样式表(英文全称：Cascading Style Sheets)是一种用来表现HTML（标准通用标记语言的一个应用）或XML（标准通用标记语言的一个子集）等文件样式的计算机语言。

不知道您有没有遇到过这种情况：在打开网页时所有的网页内容都是左对齐、字体都相同并且其他网页内容也没有原本的样式，这是因为网页的CSS文档缺失，可以看出CSS的作用就是控制网页的样式。

为什么使用CSS

  • 通过CSS，将表示网页的内容和样式的代码分离；
  • 解决了html代码对于样式定义的重复，提高了代码复用性；
  • 增强了网页的实现效果。

CSS规则

CSS样式表包含三部分内容：选择符、属性和属性值。语法格式为选择符{属性:属性值;}

  • 选择符：又称选择器，对网页的样式控制就是通过不同的选择器实现的；
  • 属性：主要包括字体属性、文本属性、背景属性、布局属性、列表项目和表格属性等；
  • 属性值：为某属性的有效值。属性与属性值之间用“:”分隔，如果有多个属性值，使用“;”分隔。

CSS选择器

CSS常用的选择器有标记选择器、类别选择器和ID选择器。

标记选择器

HTML代码通过不同的标签来标记不同的网页内容，比如<a>和<img>分别用于标记超链接和图片。标记选择器可以对不同的html标签进行样式控制。

类别选择器

如果想对一个网页中相同标签设计不同的样式，使用标记选择器就无法实现了，这个时候使用类别选择器更为恰当。

  • 类别选择器与html标签结合时不管标签种类，只管标签class属性的属性值。

ID选择器

ID选择器通过HTML标签的id属性和属性值来更改样式，与类别选择器基本相同，需要注意的是，一个HTML页面中不能有两个相同的ID标记，因此ID选择器只能被使用一次。

  • ID选择器语法#id{属性:属性值;};
  • html标签的id属性一般用于标记网页上特殊的一部分区域。

由于CSS代码是用于控制网页样式的，必须在网页内容之前加载，所以<style>一般在<head>标签内部（包括后面的导入、链接等方式都是在<head>标签内进行的）。
