MySQL 最小自定义安装教程

发表于 2018-10-13 |

简介

MySQL 是一个轻量级的数据库管理软件，在轻量级 Web 应用程序应用广泛，然而由于 MySQL 的全英安装界面和并不人性化的设计，要完全正确安装并不容易，本篇介绍如何在 Windows 操作系统上进行 MySQL 的最小自定义安装。

  • 最小：仅安装学习过程会使用到的软件；
  • 自定义：MySQL 离线安装版会默认将全部组件安装在 C 盘，本文介绍了一种自定义安装路径的方法。

安装 MySQL

本文使用官方提供的 mysql-installer-community-8.0.11.0 来进行安装。

运行 mysql-installer-community-8.0.11.0.exe 开始安装，下面依次介绍安装过程中需要注意的步骤。

选择需要的组件和安装位置

第一步为同意用户协议，勾选同意之后点击 next 进入下一步选择安装方式。

安装方式介绍：

  • Developer Default：默认的开发人员方式，会将以下组件安装在 C:\Program Files 和 C:\Program Files(x86)

    • MySQL Server：MySQL 服务器，也是我们学习使用的最主要工具；
    • MySQL Shell：一个 MySQL 脚本软件；
    • MySQL Router：一个可以提高应用程序和数据库交互速度的组件；
    • MySQL Workbench：MySQL 的图形化操作界面；
    • MySQL Connectors：MySQL 驱动，如果你的电脑上安装了 Java 或 Python，该组件的内容就是 JDBC 或 Python DB-API；
    • Examples and tutorials 和 Documentation：演示、教程和文档。

  • Custom：自定义安装方式，Developer Default 方式安装的部分组件对于我们来说可能使用不到，可以通过自定义方式将我们需要的功能安装到指定的位置。我们选择如下组件：MySQL Server、MySQL Workbench、MySQL Router 、 Examples and tutorials 和 Documentation（MySQL Router 是 Workbench 以及其他操作数据库的应用程序的必须组件）。

左侧 Available Products 为可用组件，右侧为已选组件。将需要的组件选择后依次为每个组件选择安装位置：

依次点击组件、Adcanced Options，在弹出的窗口中选择位置。

  • MySQL Server 需要选择两个安装位置：
    • Install Directory 安装位置和 Data Directory 数据位置，我分别将其选择在 MySQL Server 和 MySQL Data 两个文件夹下
  • 依次选择好所有组件的安装目录之后进入下一步，如果之前出现过黄色的警告，那么现在会多出一步解决路径冲突的问题，这是因为 MySQL 不希望你将组件安装在一个非空的文件夹，如果您确认警告中的路径为一个空文件夹直接无视即可。

开始安装

点击 Execute 开始安装即可：

注：在安装步骤之前可能会有 check requiremens 检查运行环境的步骤，会列出您电脑中缺少的组件，需要安装运行环境之后才能继续。

基本配置

安装完成之后进行一些 MySQL 的基本配置，其他选项都不用改变直接 next 即可，在为 root 用户创建密码时输入您期望的密码即可。

创建好密码之后再点几次 next 进入应用配置界面，点击 Execute 开始执行即可。

到此安装全部完成，但是仍有一些配置性工作，本文不再赘述。

MySQL Installer 是 MySQL 提供的管理工具，在第一次点击 MySQL 安装包时自动安装，您可以在任何时候使用该工具来为您的 MySQL 添加或删除组件。
