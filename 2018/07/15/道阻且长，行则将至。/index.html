道阻且长，行则将至。

发表于 2018-07-15 | 分类于 随便聊聊 |

一不留神大学生活已经过去了一半，难免感触不少，于是写一篇对于大学生活前两年的总结。

请允许我先自我介绍一下，我是一名双非院校的物联网工程专业二年级在读学生，一个潜在的码农，一个“中等白”——毕竟自身水平还差的很远，称不上大白，但是又不想称作小白以至于否定了这两年的努力，于是用了“中等白”这个词来描述自己，希望有朝一日可以更进一步。

大一

网瘾少年的自我救赎、国防教育拓展班、亲爱的女朋友

2016 年 6 月 7 日下午，没有想象中的激动、也没有传说中剑客收刀入鞘的骄傲，只有一丝的无所适从和更多的网瘾——以至于我高考完第一天就在网吧玩到了凌晨两点。辛亏上大学买的笔记本电脑很渣，辛亏小牛村的特殊地理环境，不然大一一整年大多数时候我都会像那个暑假一样沉迷在各种游戏里。

十五年前家父是西北一个小县城的初中物理老师兼任计算机老师，于是在那个初中生上计算机课需要套鞋套洗手的年代，还没上小学的我已经学会了他们上课要讲的内容：开关机、打开关闭文件和文件夹，甚至画图。辛亏这从小培养的兴趣，我在大一某一个忍受不了高延迟的游戏的晚上决定自学编程，记忆尤新的是，将网课视频中的 C 语言 Hello World 代码正确抄进 VC 6.0 并且编译运行就花了我 五六个小时。

后来加入了学校里的几个学术科技类社团，参加了几次培训活动、加上看网课等，虽然磕磕绊绊，但是也算是慢慢步入正轨。到了大一下学期，阴差阳错加入一个学科竞赛小组，大家第一次碰面的时候分配工作，学长们问我想做什么，我支支吾吾半天说了句我现在还什么都不会，想学 Java 。现在想想，当时那个竞赛项目完全用不到 Java 程序员，也不需要一个菜鸟都算不上的小白，感谢学长没有把我踢出项目组，甚至不需要我关心项目进度，这个经历让我收获颇丰，甚至于后来接手了这个社团。

不过我的大一更重要的两个关键字是国旗班和女朋友。

为了仅有的几次能在全校人面前穿海军礼服升国旗，我加入了学校的国防教育拓展班国旗队，大多数时候都是早上五点半起床六点十分开始训练、每周四晚练体能，虽然训练很累，但是加入国防教育拓展班的经历绝对是我最怀念的。

大一第二学期初，我和现在的女朋友确定关系，这是我二十年人生做的最正确的一个决定。她是我的高中同学，是一个笑起来很可爱而且性格很好的女生，不知道什么时候起悄悄走进了我的心里。不过由于一些不想再提的原因，我真正向她示爱的时候已经是大一第一学期结束了，到现在在一起已经 506 天了，虽然异地很辛苦，但是我和她都在很用心经营这段感情，我一直坚信我们最后一定能有个好的结果。

大二

物联网协会

说出来你们可能不信，我的 QQ 空间应该是最精彩的那一类了。有一个在大二第一学期拿下 CCIE 的大佬，有一个国家级黑客大赛拿下全国第一的大佬，有一对在 520 当天一起拿到学科竞赛奖项的情侣（我能想到最浪漫的事就是和你一起拿奖？），有一个擅长英语、人美声甜、校级朗诵比赛拿奖的小姐姐，还有一个精通摄影的小姐姐。诸如此类，见识到其他人有多么优秀的时候我终于开始不断审视自我，开始有计划、有目的的学习，于是这一年我放弃了学生会和国旗班，只留下了一个学术科技类社团——物联网协会。

物联网协会是我大一时参加培训次数最多的社团，不知道我何德何能最后居然成了新一任的会长。在担任会长的这一年里，我为新加入社团的大一新生讲基础的 C++ 语法、说我们曾经踩过的坑、介绍优秀的 IT 论坛等，在给新生进行培训时，我逐渐改掉了一些以前性格上的弱点，也深刻认识到自己的技术水平还远远不够。在每年一度本科生创新计划开始的时候，身为社团会长的我理所当然的成了社团申报的项目的负责人，不止要解决技术问题、还要处理一些文案、包括给项目组成员分工等工作对我来说都是新的难题，也都在促使我慢慢进步。

今后

道阻且长，行则将至。

这是下一个学期的小计划，发出来和大家交流、共同进步，希望半年后再回头总结时我不会对自己失望。

  • 学习数据结构
  • 每周至少一道 LeetCode
  • 完成自己的第一款 Web 作品
  • 加强 JS ，学习微信小程序开发
  • 能流畅使用 Linux 系统
  • 大三第一学期绩点高于 3.0
  • 每周至少用百词斩背三天单词
  • 每周至少跑一次三千米
