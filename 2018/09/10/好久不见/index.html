好久不见

发表于 2018-09-10 | 分类于 随便聊聊 |

断更

从博客创建至今 276 天，发布的文章仅有寥寥二十三篇，其中还断更近四个月。虽然一直都有写学习笔记的习惯，最勤奋的时候也有每周三篇的产量，但是自认为水平不足，发布的仅有四分之一左右，都是自认为通俗易懂且严谨的文章。

没曾想发布的这些竟成了我唯一保留下的学习笔记——没错之前在学习 Linux 操作系统时执行了那个充满破坏力的命令，发现后立刻中止但是为时已晚，全部的学习笔记、很多学习资料全没了，幸好博客的本地文件还在，不然我怕是连写博客的动力都没有了。

更名

以前博客名为 CnMarijuana，Cn 表示中国，单词 Marijuana 直译罂粟、大烟，这与一个被人从初中叫到高中的外号有关（请不要误会，我是一个遵纪守法交了七年团费的共青团员。虽然我抽烟但是这个外号与我抽烟无关，这一点上总是被人误会，大多数时候我也懒得解释）。

现更名为 nullptr ，这是 C++11 中加入的新的关键字，表示指针的未初始化状态。本想使用 Java 中的关键字 null 来起名，但是在博客的配置文件 .yml 中 null 也是一个关键字，于是使用 nullptr。

对于一个引用，如果使用得当可以大大减少代码量，如果未初始化也可能导致异常，并且理论上有些引用可以指向任何 Object 。

对于我，漫漫人生路还很漫长，以后到底是为发展社会主义做出贡献还是成为社会渣滓还没有定论，以后到底从事什么工作也没有定论，使用 null 来描述我当前的状态好像也勉强说得过去。

广告

你有一个苹果，我有一个苹果，彼此交换一下，我们仍然是各有一个苹果；但你有一种思想，我有一种思想，彼此交换，我们就都有了两种思想，甚至更多。

这是一个知识爆炸的时代，获取学习资源的门槛变得很低。但是人与人之间总是有信息差，有人开始拿信息差赚钱，比如在某智播客上免费的 Java SE 学习资源，在某些淘宝店可以卖二三十，还要打上“内部专享”、“全网最低价”的字样，因为特色国情，有些免费下载的软件和源码能在 CSDN 上开出一二十块钱的价格。如果是花钱买，说不定还能图个省心，避免一些流氓软件的侵害；可要是去那些所谓的下载站，可能一不留神电脑就要上演几大安全厂商决战光明顶。

我相信每个人都有看到一些优质资源保存下来的习惯，那么为何不尝试分享这些东西，无论是资源还是经验，总有人需要它。
