Java 集合框架概述

发表于 2018-05-18 | 分类于 Java |

概述

集合是 Java 中比较重要的内容，是用于储存的容器，实质上是一套用于储存对象的容器。集合框架中的类总结有以下几个特点：

  1. 储存对象的容器；
  2. 集合长度是可变的；
  3. 集合中不能储存基本数据类型；
  4. 所有集合类都位于 java.util 包下。

集合框架

由于集合容器内部的数据结构不同，有多种集合容器，抽取共性后就形成了集合框架，而集合框架的顶层就是 collection 或 Map 接口。下面是一张集合框架的示意图：

  • 最顶层的 Iterable 是一个迭代器接口，也是 Java 集合框架中比较重要的内容，后面也会详细介绍。
  • Collection 和 Map 两个顶层接口；
  • 代表不同数据类型的子接口 List、Set 和 Queue；
  • 多个实现类，分别是对不同接口的实现，是我们要学习的主要内容。
